🍃 LeafLens AI
AI-Powered Crop Disease Detection App

About
LeafLens AI is an intelligent web application that detects 
crop diseases from leaf images using deep learning.

Features
- 🔍 Detects 38 crop diseases instantly
- 📷 Upload photo or use camera directly
- 💊 Treatment and prevention recommendations
- 📱 Installable as mobile app (PWA)
- 🌿 Supports: Tomato, Corn, Grape, Potato, Apple and more

Tech Stack
- Python Flask
- TensorFlow / Keras (MobileNetV2)
- PlantVillage Dataset (38 classes)
- HTML, CSS, JavaScript
- Progressive Web App (PWA)

How to Run
1. Clone this repo
   git clone https://github.com/ER-PRANAV001/leaflens-ai.git
2. Install requirements
   pip install -r requirements.txt
3. Run app
   python app.py
4. Open browser
   http://127.0.0.1:5000
